# Premium Selection — главная страница

Корпоративный сайт-хаб группы компаний **Premium Selection Group**.
3 направления: Marketing & Research / Production / IT.

---

## Стек

- HTML / CSS / Vanilla JS — без сборки
- Шрифты: Commissioner + JetBrains Mono (Google Fonts)
- Палитра: Pantone 485 (#DA291C) + чёрный + белый
- Адаптив: desktop / tablet (≤1280px) / mobile (≤900px)

---

## Структура

```
.
├── index.html        — главная (hero 8×6 + 6 секций + футер)
├── discover.html     — ТЗ-генератор (8 шагов → PDF)
├── robots.txt        — поиск (discover закрыт от индексации)
├── sitemap.xml       — карта сайта
├── vercel.json       — security headers + cache (для Vercel)
├── _headers          — то же для Netlify
├── images/           — фото-мокапы продукции (1.jpg … 15.jpg, 8.jpg намеренно пропущен)
└── README.md
```

---

## Запуск локально

Просто открой `index.html` в браузере. Никаких билдов.

Если хочешь dev-сервер с авто-рефрешем:

```bash
# Любой статический сервер. Например:
npx serve .
# или
python3 -m http.server 8000
```

## Деплой

### Vercel (текущий хостинг)

1. [vercel.com](https://vercel.com) → Import Git Repository → выбрать этот репо
2. Framework Preset: **Other**, Build / Output: оставить пустыми
3. `vercel.json` уже в репо — security headers и cache применятся автоматически

### Netlify (альтернатива)

1. [netlify.com](https://netlify.com) → войти через GitHub
2. **Add new site → Import from Git** → выбрать этот репо
3. Build command: оставить пустым
4. Publish directory: `.` (корень)
5. Готово — каждый push в `main` → автодеплой за 20 сек

URL: `xxx.netlify.app` (можно переименовать в настройках)

### Glitch (для редактирования в браузере)

1. [glitch.com](https://glitch.com) → войти через GitHub
2. **New Project → Import from GitHub** → URL этого репо
3. Редактируешь файлы — превью обновляется мгновенно
4. **Tools → Git → Push to GitHub** — синхронизация обратно

URL: `xxx.glitch.me`

## Редактирование контента

- **Тексты** — прямо в `index.html`. Все строки — на русском, найти через Ctrl+F
- **Цвета** — переменные в `:root` (`--red`, `--ink`, `--paper`)
- **Шрифты** — `--ff-display`, `--ff-text`, `--ff-mono`
- **Размеры** — шкала `--fs-display`, `--fs-body-l`, `--fs-body`, `--fs-meta`, `--fs-cap`
- **Контакты направлений** — секция `.contact-grid` (3 карточки)
- **Фото в hero и кейсах** — заменять файлы в `images/` (одинаковые имена)

## Что внутри

### Hero
Грид-коллаж 8×6 с разными размерами блоков, фото-мокапы циклически меняются. Центр-текст: «Мы / Premium Selection / создаём грандиозные проекты». Слово «грандиозные» крутится по синонимам.

### 02. Направления
Чёрная секция с 3 строками-карточками. На hover полоса заливается красным.

### 03. О группе
Белый блок с манифестом и описанием.

### 04. Цифры
Красная полоса с 4 метриками (лет на рынке, проектов, стран, направлений).

### 05. Работы
Белая секция с 7-карточным грид-коллажем кейсов.

### 06. Контакты
Чёрная секция, 3 карточки направлений с телефонами/email/менеджерами.

### Футер
Бренд + 3 колонки навигации + копирайт.
